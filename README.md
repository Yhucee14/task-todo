# Interactive Task Card

A clean, high-fidelity, fully interactive Task Card component built for a modern productivity app.

## Features

- All required `data-testid` attributes for automated testing
- Real checkbox with proper semantics
- Live time remaining updates every 30 seconds
- Beautiful due date & time formatting ("Due in 3 days", "Overdue by 2 hours", etc.)
- Fully editable via modal
- Hover animations and micro-interactions
- Responsive and accessible
